## Setup 

> You need **Node Js, Mongo Db & NPM** installed

<u><b>Install dependencies</b></u>

```bash
$ npm install
# Or Yarn
$ yarn install
```

<u><b>Create config</b></u>

create a file `config/config.env`

and fill it with

```
PORT = LOCAL_PORT
MONGO_URI = MONGO_DB_URL

GOOGLE_CLIENT_ID = GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET = GOOGLE_CLIENT_SECRET
```

Check <a href="https://www.youtube.com/watch?v=SBvmnHTQIPY&t=2037s">InÂ video</a>

<u><b>Run server</b></u>

> run server with `start` or `dev`

```bash
$ npm run dev
## or yarn
$ yarn dev
```
